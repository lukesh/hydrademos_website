WebORB Presentation Server
 
 version 3.6.0.3 - January 2009		  

 

What's new in 3.6

WebORB for .NET is a multi-protocol presentation server that is to designed to effortlessly connect AJAX, Flex, Flash and Silverlight clients with .NET objects, XML Web Services, native operating system resources and real-time messaging streams. Using WebORB, client applications can communicate with each other or with remote server-side objects and services as if they are local in the client environment.

To put things in perspective, WebORB resides in a web server between your .NET class libraries or external XML Web Services and rich client applications. It is a gateway or Object Request Broker (ORB) between rich clients and the backend systems that understands Flex and Flash client protocols. For example, Flex applications can use new AMF3 protocol and the RemoteObject tag/API to communicate with .NET; Flash clients can use Macromedia Flash Remoting Components to connect to any object/service available through WebORB; and JavaScript/AJAX clients can use Rich Client System JavaScript library included with this WebORB distribution. Additionally, the Enterprise Edition provides a native .NET implementation of the RTMP protocol and thus supports Flex Messaging clients as well as standard Flex and Flash messaging applications.
This management console is a good starting point to become familiar with the product; it currently provides an interface for the most common product functions including: server configuration, service deployment and management as well as product licensing.
